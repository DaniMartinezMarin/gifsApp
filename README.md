# GifsApp

This project is built on Angular and allows you to consume a Gifs API.
It consists of a search bar in which you can write the name of the gif to search for. On the left is a history of the searches carried out, in which you can click and show those results again.

You can see a demo at:
https://clinquant-zuccutto-b5a3e8.netlify.app/
